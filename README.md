A structured, multi-page educational website presenting psychological concepts,
developed as part of the Informatics certification project.
The focus was on creating an intuitive navigation experience across multiple pages,
with a consistent layout and integrated media content to enhance the learning experience.

Features:
- Multi-page website with smooth navigation between sections
- Consistent design and layout across all pages using CSS
- Integrated image gallery for visual learning
- Embedded video content for richer educational material
- Modular folder structure for styles, scripts, images, and videos

Technologies:
- HTML5: semantic page structure
- CSS3: styling, layout, and visual design
- JavaScript: interactivity and dynamic elements
